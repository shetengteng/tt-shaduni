# Typography

## Usage

```vue
<template>
  <tt-typography />
</template>
```

## Props

| Prop | Type | Default |
|---|---|---|
| `type` | `TypographyType` | `'text'` |
| `level` | `TypographyLevel` | `3` |
| `ellipsis` | `boolean` | `false` |
| `bold` | `boolean` | `false` |
| `underline` | `boolean` | `false` |
| `delete` | `boolean` | `false` |
| `disabled` | `boolean` | `false` |
| `href` | `string` | `''` |
