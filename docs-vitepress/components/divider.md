# Divider

## Usage

```vue
<template>
  <tt-divider />
</template>
```

## Props

| Prop | Type | Default |
|---|---|---|
| `direction` | `'horizontal' | 'vertical'` | `'horizontal'` |
| `type` | `'horizontal' | 'vertical'` | `'horizontal'` |
| `default` | `any` | `'horizontal'` |
| `contentPosition` | `'left' | 'center' | 'right'` | `'center'` |
| `type` | `'left' | 'center' | 'right'` | `'center'` |
| `default` | `any` | `'center'` |
