# Swiper

## Usage

```vue
<template>
  <tt-swiper />
</template>
```

## Props

| Prop | Type | Default |
|---|---|---|
| `autoplay` | `boolean` | `false` |
| `interval` | `number` | `3000` |
| `duration` | `number` | `500` |
| `circular` | `boolean` | `true` |
| `indicatorDots` | `boolean` | `true` |
| `indicatorColor` | `string` | `'rgba0` |
| `indicatorActiveColor` | `string` | `''` |
| `current` | `number` | `0` |
