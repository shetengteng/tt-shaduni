# CountDown

## Usage

```vue
<template>
  <tt-count-down />
</template>
```

## Props

| Prop | Type | Default |
|---|---|---|
| `time` | `number` | `0` |
| `type` | `number` | `0` |
| `default` | `any` | `0` |
| `format` | `string` | `'HH:mm:ss'` |
| `type` | `string` | `'HH:mm:ss'` |
| `default` | `any` | `'HH:mm:ss'` |
| `autoStart` | `boolean` | `true` |
| `type` | `boolean` | `true` |
| `default` | `any` | `true` |
