# Pagination

## Usage

```vue
<template>
  <tt-pagination />
</template>
```

## Props

| Prop | Type | Default |
|---|---|---|
| `modelValue` | `number` | `1` |
| `totalItems` | `number` | `0` |
| `itemsPerPage` | `PaginationMode` | `10` |
| `mode` | `PaginationMode` | `'number'` |
| `showPageSize` | `number` | `5` |
