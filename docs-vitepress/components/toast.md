# Toast

## Usage

```vue
<template>
  <tt-toast />
</template>
```

## Props

| Prop | Type | Default |
|---|---|---|
| `message` | `string` | `''` |
| `type` | `string` | `''` |
| `default` | `any` | `''` |
| `type` | `'text' | 'success' | 'fail' | 'loading'` | `'text'` |
| `type` | `'text' | 'success' | 'fail' | 'loading'` | `'text'` |
| `default` | `any` | `'text'` |
| `duration` | `number` | `2000` |
| `type` | `number` | `2000` |
| `default` | `any` | `2000` |
| `show` | `boolean` | `false` |
| `type` | `boolean` | `false` |
| `default` | `any` | `false` |
