# List

## Usage

```vue
<template>
  <tt-list />
</template>
```

## Props

| Prop | Type | Default |
|---|---|---|
| `loading` | `boolean` | `false` |
| `finished` | `boolean` | `false` |
| `loadingText` | `string` | `'Loading...'` |
| `finishedText` | `string` | `'No more data'` |
| `offset` | `number` | `300` |
