# DatePicker

## Usage

```vue
<template>
  <tt-date-picker />
</template>
```

## Props

| Prop | Type | Default |
|---|---|---|
| `modelValue` | `DatePickerMode` | `''` |
| `mode` | `DatePickerMode` | `'date'` |
| `title` | `string` | `''` |
| `show` | `boolean` | `false` |
| `minDate` | `string` | `''` |
| `maxDate` | `string` | `''` |
