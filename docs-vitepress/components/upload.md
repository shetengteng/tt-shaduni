# Upload

## Usage

```vue
<template>
  <tt-upload />
</template>
```

## Props

| Prop | Type | Default |
|---|---|---|
| `fileList` | `UploadFile[]` | ` => []` |
| `maxCount` | `number` | `9` |
| `maxSize` | `number` | `10 * 1024 * 1024` |
| `accept` | `string` | `'image'` |
| `disabled` | `boolean` | `false` |
| `previewImage` | `boolean` | `true` |
| `deletable` | `boolean` | `true` |
