# ScrollView

## Usage

```vue
<template>
  <tt-scroll-view />
</template>
```

## Props

| Prop | Type | Default |
|---|---|---|
| `scrollX` | `boolean` | `false` |
| `scrollY` | `boolean` | `true` |
| `refresherEnabled` | `boolean` | `false` |
| `refresherTriggered` | `boolean` | `false` |
| `lowerThreshold` | `number` | `50` |
