# Slider

## Usage

```vue
<template>
  <tt-slider />
</template>
```

## Props

| Prop | Type | Default |
|---|---|---|
| `modelValue` | `number` | `0` |
| `min` | `number` | `0` |
| `max` | `number` | `100` |
| `step` | `number` | `1` |
| `disabled` | `boolean` | `false` |
| `activeColor` | `string` | `''` |
| `inactiveColor` | `string` | `''` |
