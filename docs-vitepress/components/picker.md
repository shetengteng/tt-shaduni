# Picker

## Usage

```vue
<template>
  <tt-picker />
</template>
```

## Props

| Prop | Type | Default |
|---|---|---|
| `modelValue` | `string | number | (string | number)[]` | `''` |
| `columns` | `Array<{ text: string; value: string | number; children?: any[] }` | ` => []` |
| `title` | `string` | `''` |
| `confirmText` | `string` | `'Confirm'` |
| `cancelText` | `string` | `'Cancel'` |
| `show` | `boolean` | `false` |
