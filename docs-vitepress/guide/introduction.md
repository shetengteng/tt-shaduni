# Introduction

TT ShadUni is a component library for UniApp built with Vue 3, inspired by the [shadcn/ui](https://ui.shadcn.com/) design aesthetic.

## Features

- **60+ Components** covering forms, navigation, feedback, layout, and data display
- **Multi-Platform** — H5, WeChat Mini-program, Alipay Mini-program, App
- **Dark Mode** — CSS variable based theming with instant toggle
- **TypeScript** — Full type definitions for all components
- **Tree Shaking** — On-demand import, only bundle what you use
- **Accessible** — Semantic HTML and ARIA attributes where applicable
- **easycom** — Auto-register components with zero config

## Design Principles

1. **Minimal** — Clean, neutral aesthetic that works in any context
2. **Composable** — Small, focused components that compose well together
3. **Customizable** — CSS variables for full theme control without rebuilding
4. **Platform-aware** — Uses `px` units and avoids platform-specific APIs where possible
